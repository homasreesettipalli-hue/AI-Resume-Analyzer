# AI Resume Analyzer

## Overview
AI Resume Analyzer is a Streamlit-based application that compares a resume with a job description and provides an ATS-style match score.

## Features
- Upload Resume PDF
- Extract Resume Text
- ATS Match Score
- Skill Matching
- Missing Skills Detection
- Personalized Suggestions

## Technologies Used
- Python
- Streamlit
- PyPDF2
- Scikit-learn
- Pandas

## How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
```
